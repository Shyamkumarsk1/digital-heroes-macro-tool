# Smart Macro Calculator & Meal Prep Planner 🥗

**Built for the Digital Heroes Developer Trial Task**

A lightning-fast, 100% client-side web application designed to calculate personalized daily macros and instantly generate a clean, printable meal prep cheat-sheet. 

### 🚀 Live Demo
https://digital-heroes-macro-tool.vercel.app/

### 💡 The Problem it Solves
Fitness apps are often overly complex, requiring sign-ups, subscriptions, and daily logging. I built this tool because users often just want a simple, one-time calculation that produces a static, printable guide to stick on their fridge for weekly meal prep.

### 🛠️ Technical Implementation
To ensure maximum speed and zero dependencies, this tool was built without heavy frameworks or external libraries.
* **Architecture:** Single Page Application (SPA) utilizing a custom "Two-View" DOM manipulation strategy to toggle between the calculator and the final output sheet without page reloads.
* **Tech Stack:** Vanilla HTML5, CSS3, and ES6 JavaScript.
* **Export Logic:** Utilizes native browser APIs and strict `@media print` CSS queries to cleanly format the DOM for PDF printing, ensuring a perfect export across all devices without relying on brittle third-party PDF libraries.
